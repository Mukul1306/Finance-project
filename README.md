# SRM Finance Management System

A full-stack MERN Finance Management Platform designed for:

* Society Finance Management
* Daily Saving Management
* Loan Management
* Agent Collection System
* Expense Tracking
* Profit & Loss Reporting

---

## Features

### Society Module

* Create Society
* Manage Members
* Monthly Installment Collection
* Penalty Management
* Payment Tracking
* Reports
* Profit & Loss

### Loan Module

* Create Loan
* EMI Collection
* Interest Collection
* Loan Closure
* Loan Reports

### Daily Saving Module

* Agent Management
* Area Group Management
* Daily Member Registration
* Daily Collection
* Auto Penalty Calculation
* Daily Loan System
* Notifications
* Reports
* Profit & Loss

### Agent Portal

* Agent Login
* Assigned Members
* Daily Collection
* Loan EMI Collection
* Collection History
* Dashboard Analytics

---

## Tech Stack

### Frontend

* React.js
* React Router
* Axios
* Tailwind CSS
* Lucide Icons

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

### Database

MongoDB Atlas

### Deployment

Backend: Render

Frontend: Netlify / Vercel
